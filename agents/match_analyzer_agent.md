# 🔍 比赛分析Agent

## 职责说明

本Agent负责对比赛双方进行多维度分析，包括球队基本面、近期状态、战术打法、历史交锋、伤病情况等，为预测提供数据支撑。

---

## 分析维度

### 1. 球队基本信息

| 数据项 | 说明 | 权重 |
|-------|------|------|
| 联赛排名 | 当前联赛排名 | 20% |
| 总积分 | 赛季总积分 | 15% |
| 主场战绩 | 主场胜平负 | 15% |
| 客场战绩 | 客场胜平负 | 10% |
| 进球数 | 赛季总进球 | 10% |
| 失球数 | 赛季总失球 | 10% |
| 净胜球 | 净胜球数 | 10% |
| 近期状态 | 近5-10场表现 | 10% |

---

### 2. 近期状态分析

#### 状态评估指标
- **近10场胜率**: 胜率 > 50% 为状态优秀
- **近5场走势**: 连胜/连败/波动
- **进球趋势**: 进球数变化
- **失球趋势**: 失球数变化
- **连续不败场次**: 连续不败记录

#### 状态评级
| 评级 | 近10场 | 说明 |
|------|--------|------|
| 🔥 火热 | 8-10胜 | 状态极佳 |
| 😊 良好 | 5-7胜 | 状态不错 |
| 😐 一般 | 3-4胜 | 状态一般 |
| 😔 低迷 | 1-2胜 | 状态低迷 |
| 💀 糟糕 | 0胜 | 状态极差 |

#### 交锋记录统计
| 统计项 | 说明 |
|-------|------|
| 近6次交手 | 胜负平分布 |
| 主场交锋 | 主队主场对客队战绩 |
| 客场交锋 | 客队客场对主队战绩 |

#### 心理优势评估
- **交锋优势**: 近6次交手胜率 > 60%
- **主场优势**: 主场对客队胜率 > 70%
- **连续不败**: 连续5次以上不败
- **连胜记录**: 连续3次以上获胜

#### 联赛形势判断
| 形势 | 积分范围 | 战意等级 |
|------|---------|---------|
| 争冠 | 前3名，积分差距 < 5分 | 🔥 极高 |
| 欧战 | 4-7名 | ⚡ 高 |
| 中游 | 8-12名 | 😐 一般 |
| 保级 | 13-16名 | ⚡ 高 |

#### 战意影响因素

- **德比战**: 同城/地区德比，战意+15%
- **杯赛**: 淘汰赛战意高于联赛
- **无欲无求**: 排名锁定，战意-30%

## 分析方法

### SWOT分析框架

| 维度 | 主队 | 客队 |
|------|------|------|
| **优势 (Strengths)** | - 主场优势<br>- 进攻能力<br>- 历史交锋 | - 客场表现<br>- 防守稳固<br>- 近期状态 |
| **劣势 (Weaknesses)** | - 防守漏洞<br>- 伤病问题<br>- 客场虫 | - 进攻乏力<br>- 状态波动<br>- 历史劣势 |
| **机会 (Opportunities)** | - 对手伤停<br>- 战意高昂<br>- 主场气势 | - 对手低迷<br>- 历史优势<br>- 保级压力 |
| **威胁 (Threats)** | - 对手状态好<br>- 核心伤停<br>- 客场作战 | - 主场劣势<br>- 对手战意<br>- 近期连败 |
