## 一、项目背景

### 1.1 背景概述

本项目 `europe_leagues` 是面向**五大联赛**（英超/西甲/德甲/意甲/法甲）及**欧战赛事**（欧冠/欧联/欧协联）的比赛预测与分析系统。系统基于多模型融合、实时赔率采集、RAG记忆检索和动态权重调整等技术，提供专业的足球比赛预测服务。

### 1.2 核心痛点

- **数据分散**: 赛程、球员、赔率数据分散在不同平台和格式中
- **预测不一致**: 单一模型预测容易过拟合，缺乏多维度交叉验证
- **复盘困难**: 赛果回填和准确率统计流程繁琐，难以形成闭环优化
- **记忆缺失**: 缺乏对历史相似盘路和爆冷案例的有效利用

### 1.3 技术演进

| 阶段 | 时间 | 核心特性 |
|------|------|----------|
| V1.0 | 2026-04 | 基础预测流程，单一模型 |
| V2.0 | 2026-05 | 10模型融合、RAG记忆层、动态权重、复盘链路 |

## 二、项目目标

### 2.1 核心目标（可量化）

| 目标 | 指标 | 当前状态 |
|------|------|----------|
| 胜平负准确率 | ≥ 45% | 38.3% (18/47) |
| 比分Top3命中率 | ≥ 20% | 17.95% (7/39) |
| 大小球准确率 | ≥ 65% | 65.62% (21/32) |
| 预测生成时效 | ≤ 5分钟/场 | 已实现 |
| 赛果回填自动化 | ≥ 80% | 已实现 |

### 2.2 功能目标

1. **稳定生成**未来N天五大联赛预测报告（默认3天，可扩展）
2. **双路径写回**: 五大联赛写`teams_2025-26.md`，欧战写`MEMORY.md`
3. **RAG记忆沉淀**: 自动检索相似盘路和爆冷案例
4. **准确率闭环**: 赛后自动回填，刷新胜负/比分/大小球统计

**3.1 10模型融合预测**

| 模型 | 权重 | 说明 |
|------|------|------|
| Poisson基础模型 | 15% | 基于进球期望的泊松分布 |
| Dixon-Coles模型 | 15% | 考虑进球相关性的改进泊松 |
| Elo评分模型 | 10% | 基于球队历史实力评分 |
| 近期状态模型 | 15% | 近5场战绩加权 |
| 主客场模型 | 10% | 主客场胜率差异 |
| 进球期望模型 | 10% | xG数据驱动 |
| 赔率隐含模型 | 10% | 从赔率反推概率 |
| 历史交锋模型 | 5% | 直接对话记录 |
| 战意模型 | 5% | 保级/争冠动力 |
| 伤病影响模型 | 5% | 关键球员缺阵影响 |

**3.2 RAG记忆检索**

```python
# RAG检索流程
1. 构建查询向量（球队、盘口、联赛特征）
2. 从rag_store检索相似比赛（Top 5）
3. 分析相似盘路结果分布
4. 生成retrieved_memory_explanation
```

**3.3 爆冷风险分析**

```python
# 爆冷分析流程
1. 查询爆冷案例库
2. 计算风险评分(0-100)
3. 识别风险因子
4. 生成风险提示
```

**风险等级**:
- 极低(0-20): 🟢 正常预测
- 低(21-40): 🟢 略防冷门
- 中(41-60): 🟠 注意防范
- 高(61-80): 🔴 强烈防范
- 极高(81-100): 🔴 可能大冷

**3.1 单场复盘**

每场比赛生成复盘结论:
```
比赛: 塞尔塔 2-3 莱万特
预测: 主胜 (38.3%) | 比分: 1-0/2-0 | 大小: 小2.75 (60.6%)

复盘结论: 胜平负未中 比分未中 大小球未中
```

**3.2 滚动记忆复盘总结**

基于最近N场已完赛样本生成总结:
```markdown
### 滚动记忆复盘总结（最近 8 场已完赛）

> 基于滚动记忆区块中的 8 场已完赛样本生成。当前结论：近期样本回撤明显，需要同时复查方向判断、比分排序和总进球弹性。

#### 总体复盘

- 已完赛 `8` 场，胜平负命中 `4/8`，命中率 `50.0%`
- 比分命中 `1/8`，命中率 `12.5%`
- 大小球命中 `4/4`，命中率 `100.0%`
- 样本分布：主胜4场 / 平局1场 / 客胜3场

#### 单场复盘

- `朗斯 0-2 巴黎圣日耳曼`
  赛前预测为 `主胜`，最终赛果为 `客胜 0-2`。
  复盘结论：胜平负未中 比分未中 大小球命中
  主方向判断失真，需要回看盘口变化、临场信号和样本相似性。
...

#### 优化方向

- 胜平负：回看近期未命中样本中的盘口退让、平赔下修和临场结构
- 比分：增强 `2-0`、`3-0`、`3-1` 一类强队兑现型比分的排序权重
- 大小球：提高开放局、强队主场和高节奏联赛场景的总进球弹性
- 风险提示：当胜平负方向成立但总进球预期偏保守时，明确提示大小球存在上修风险
```
