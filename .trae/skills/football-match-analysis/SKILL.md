# Football Match Analysis

本Skill是足球比赛预测分析系统的核心技能，集成多维度数据分析、机器学习模型和博彩数据研究，提供专业的比赛预测和投注建议。

## 项目架构集成

本技能与项目的专业Agent系统深度集成：

| Agent | 功能 | 集成方式 |
|-------|------|----------|
| 数据采集Agent | 从竞彩网、澳客等权威网站获取实时赔率数据 | 通过browser-use/playwright自动抓取 |
| 比赛分析Agent | 球队基本面、战术打法、历史交锋分析 | 调用分析模块 |
| 赔率分析Agent | 凯利指数、盘口深度分析 | 实时数据处理 |
| 结果追踪Agent | 预测准确率统计与优化 | 持续反馈机制 |

## 核心功能

- 多联赛（欧洲五大联赛）比赛分析
- 实时赔率数据获取与分析
- 凯利指数深度研究
- 泊松分布概率预测
- 预测准确率跟踪与优化

### 数据获取方法

#### 自动获取（推荐）

1. **网页抓取**：使用 agent-browser 或 playwright-cli 访问目标页面
2. **API 接口**：调用公开的赔率数据 API
3. **实时更新**：比赛前获取最新赔率变化

#### 手动获取

1. **访问竞彩网**：`https://www.sporttery.cn/jc/zqszsc/` 
   - 查找对应比赛编号（如周五007）
   - 记录胜平负赔率、让球赔率、大小球赔率
2. **访问澳客官网**：`https://www.okooo.com/livecenter/`
   - 查看凯利指数多家公司对比
   - 分析赔率变化趋势
3. **访问500彩票网**：`https://zx.500.com/` 
   - 对比欧赔数据
   - 查看亚盘盘口变化
4. **搜索即时数据**：使用搜索引擎搜索"圣保利 科隆 赔率"获取最新数据
5. **查看首发阵容和伤病名单**
6. **检查首发阵容是否符合球队常规主力阵容**

#### 数据验证

1. 对比多个博彩公司赔率一致性
2. 检查赔率变化趋势是否合理
3. 验证盘口与球队实力是否匹配
4. 检查大小球指数是否合理

### 分析维度

### 1. 球队基本信息

- **球队实力排名**：联赛排名、总积分、主场/客场战绩
- **近期状态**：最近5-10场比赛的胜负平情况、进球/失球数
- **进球能力**：场均进球数、主场/客场进球数
- **防守能力**：场均失球数、零封场次比例

### 2. 战术打法分析

- **阵型选择**：常用阵型（4-3-3、4-4-2、3-5-2等）
- **进攻风格**：边路传中、中路渗透、长传冲吊
- **防守策略**：高压逼抢、低位防守、稳守反击
- **控球能力**：场均控球率、传球成功率

### 3. 关键球员状态

- **核心球员**：进球数、助攻数、关键传球数
- **伤病情况**：主力球员伤病、是否缺阵
- **停赛情况**：黄牌/红牌停赛
- **体能状况**：一周多赛、轮换情况

### 4. 球队战意分析

- **联赛形势**：争冠/保级/欧战资格竞争
- **战意等级**：全力以赴/轮换调整/无欲无求
- **主客场因素**：主场优势、客场虫/主场龙
- **交锋战意**：德比战/关键战特殊战意

### 5. 历史交锋记录

- **近期交锋**：最近3-5次交锋结果
- **主场客场因素**：主客场对交锋结果的影响
- **进球规律**：交锋中的进球数规律
- **盘路走势**：历史盘口表现

### 6. 环境与外部因素

#### 6.1 天气条件影响

| 天气类型 | 对比赛的影响 | 投注启示 |
|---------|------------|---------|
| 雨天 | 场地湿滑，球速快，进球可能增加；高空球优势队伍有利 | 大球概率↑ |
| 雪天 | 场地条件恶劣，失误增多；弱队可能死守 | 小球概率↑ |
| 大风天 | 传球精准度下降，长传球战术更有效 | 特定战术队伍占优 |
| 高温 | 球员体能消耗大，下半场进球往往偏多 | 关注大小球盘口 |
| 低温 | 球员热身不充分，伤病的概率提升 | 关注首发阵容 |

#### 6.2 场地与主场因素

- **场地条件**：草皮质量影响传球和控球
- **主场氛围**：狂热主场提升主队战斗力（通常+10-15%）
- **旅途因素**：长途跋涉影响客队体能（跨时区+额外损耗）
- **时差因素**：跨国比赛需要考虑时区适应

**主场优势量化参考**：
```
主场胜率提升 = 主场基础胜率 × (1 + 0.1~0.15)
联赛类型调整：
- 英超：主场优势约15%
- 西甲：主场优势约12%
- 德甲：主场优势约13%
- 意甲：主场优势约14%
- 法甲：主场优势约11%
```

#### 6.3 球队体能状况

**体能评估维度**：
- **一周多赛**：3天内第2场比赛，体能下降约15-20%
- **轮换幅度**：是否进行大规模轮换
- **关键球员体能**：核心球员是否得到充分休息
- **伤病变数**：疲劳导致的隐性伤病

**体能影响评估**：
```
体能充沛 vs 疲劳：
- 进攻效率：+15% vs -15%
- 防守稳定性：+10% vs -20%
- 下半场表现：+20% vs -25%
- 跑动距离：+12% vs -18%
```

#### 6.4 球队心态与舆论

- **舆论压力**：媒体评价对球队心理的影响
- **赛季目标**：球队对赛季结果的期望值
- **球员合同**：合同年球员往往更拼命
- **换帅效应**：新帅上任往往带来短期状态提升
- **德比战特殊心态**：德比战战意额外增强

### 7. 赔率与盘口分析

- **胜平负赔率**：转换为概率分析
- **让球盘口**：盘口合理性分析
- **大小球指数**：进球预期分析
- **庄家心理**：赔率变化背后的博弈心理

### 7.1 庄家博弈心理分析

#### 赔率类型与分析

1. **胜平负赔率（1X2）**
   - 将赔率转换为隐含概率：概率 = 1 / 赔率 × 100%
   - 对比各公司赔率差异，寻找价值投注
   - 分析赔率变化方向，判断市场共识
2. **让球盘口（Asian Handicap）**
   - 分析盘口是否合理反映球队实力差距
   - 关注盘口变化：升盘/降盘的市场含义
   - 识别诱盘行为：庄家故意调整盘口引导资金流向
3. **大小球指数**
   - 分析两队进攻/防守能力与大小球指数的关系
   - 关注大小球盘口变化
   - 结合历史交锋进球规律
   - **泊松分布分析**：使用数学模型预测进球分布
4. **首发阵容变化**
   - 分析首发阵容是否符合球队常规主力阵容
   - 关注关键球员是否缺阵
   - 考虑替补球员是否符合球队风格
5. **凯利指数（Kelly Index）**
   - 分析凯利指数判断博彩公司的赔付风险与市场信心
   - 数值越低：机构赔付压力大，怕打出（倾向看好）
   - 数值越高：机构赔付压力小，不怕打出（倾向不看好）

#### 7.1.1 凯利指数核心定义

**一句话看懂**：
凯利指数 = 赔率 × 市场平均概率，本质是衡量博彩公司对某一赛果的赔付风险。

**数值区间（实战标准）**：

| 数值区间 | 风险等级 | 市场含义 |
|---------|---------|---------|
| < 0.90 | 低风险区 | 机构高度警惕，热门/稳胆方向 |
| 0.90 ~ 1.00 | 正常区 | 无明显倾向，常规分布 |
| > 1.00 | 高风险区 | 机构不惧赔付，冷门方向 |

#### 7.1.2 澳客页面解读（关键用法）

**1. 看"一致性"（最重要）**
- 多家公司凯利集体偏低：强信号，方向稳
- 个别公司偏低、其他偏高：弱信号/诱盘，忽略

**2. 看"临场变化"（比初盘准）**
- 临场凯利持续下调：强化看好，热度可控
- 临场凯利突然拉高：热度失控/诱盘，风险大

**3. 看"离散度"（方差/离散值）**
- 方差小（<0.01）：机构高度一致 → 稳
- 方差大（>0.03）：机构分歧大 → 冷/乱

#### 6.1.3 澳客页面示例

```
       主胜    平局    客胜
威廉   0.89   0.96   1.04
立博   0.88   0.97   1.03
澳客   0.89   0.95   1.04
平均   0.89   0.96   1.04
```

**分析结论**：
- 主胜集体0.89：机构一致看好、怕赔 → 主利好
- 客胜1.04：机构不怕 → 客难出

#### 7.1.4 常见误区（一定要避）

| 误区 | 真相 |
|-----|------|
| 低凯利 = 必赢？ | 否。低凯利是热度高、机构控赔，常出稳胆/穿盘，但深盘易平局 |
| 高凯利 = 必不赢？ | 否。高凯利是机构不惧赔付，常出冷门/平局，尤其临场降凯利易冷 |
| 只看一家公司？ | 无效。必须多家对比看一致性 |

#### 7.1.5 实战口诀（直接用）

```
低一致，稳；高分散，冷；
临场降，强；临场升，坑
```

**详解**：
- **低一致，稳**：多家公司凯利同时偏低 → 机构一致防范 → 方向稳定
- **高分散，冷**：公司间凯利离散度大 → 分歧严重 → 冷门结果
- **临场降，强**：临场凯利下降 → 热度可控 → 机构真实看好
- **临场升，坑**：临场凯利上升 → 热度失控 → 诱盘风险大

#### 7.1.6 凯利指数实战分析步骤

```
Step 1: 收集多家公司凯利指数（至少3-5家）
Step 2: 计算平均值和方差
Step 3: 判断一致性：方差<0.01为高度一致
Step 4: 识别集体偏低项：凯利<0.90为热门
Step 5: 观察临场变化：判断诱盘还是真看好
Step 6: 综合庄家心理给出结论
```

#### 7.1.7 凯利指数与赔率结合分析

| 赔率类型 | 凯利指数 | 综合判断 |
|---------|---------|---------|
| 赔率低 + 凯利低 | < 0.90 | 强强联合 → 稳胆 |
| 赔率低 + 凯利高 | > 1.00 | 矛盾信号 → 诱上嫌疑 |
| 赔率高 + 凯利低 | < 0.90 | 冷门保护 → 机构防范 |
| 赔率高 + 凯利高 | > 1.00 | 冷门确认 → 可博冷 |

### 7.2 泊松分布大小球分析

#### 泊松分布原理

泊松分布是一种离散概率分布，适用于描述单位时间内随机事件发生的次数。在足球分析中，它可以用来预测进球数的概率分布。

**核心公式**：
```
P(X=k) = (λ^k × e^(-λ)) / k!
```
- `λ`：预期进球数
- `k`：进球数
- `e`：自然常数（约2.718）
- `P`：进k球的概率

#### 预期进球数计算

**基础方法**：
1. **主队预期进球** = 主队进攻能力 × 客队防守能力 × 联赛主场平均进球
2. **客队预期进球** = 客队进攻能力 × 主队防守能力 × 联赛客场平均进球

**调整因素**：
- 主场优势（通常增加10-15%）
- 近期状态（0.8-1.2的调整系数）
- 伤病情况（主力球员缺阵的影响）
- 战意因素（重要比赛的影响）

#### 大小球概率计算

1. **计算泊松分布**：为每个球队计算0-8球的概率分布
2. **构建比分矩阵**：计算所有可能比分的概率
3. **计算总进球分布**：统计总进球数的概率
4. **大小球判断**：根据盘口（如2.5球）计算大球和小球的概率

#### 分析流程

```
Step 1: 收集数据（球队进攻/防守数据）
Step 2: 计算预期进球数（λ值）
Step 3: 计算泊松分布概率
Step 4: 构建比分概率矩阵
Step 5: 分析总进球分布
Step 6: 计算大小球概率
Step 7: 生成分析报告
```

#### 模型优化

1. **Dixon-Coles模型**：修正低进球数的概率，考虑进球之间的相关性
2. **贝叶斯更新**：结合先验概率和新数据
3. **多模型融合**：结合其他分析方法
4. **实时调整**：根据比赛进程调整预测

#### 应用场景

- **赛前分析**：预测比赛进球分布
- **实时投注**：根据实时情况调整策略
- **球队评估**：分析球队进攻和防守能力
- **联赛趋势**：分析联赛整体进球趋势

#### 注意事项

- **数据质量**：模型效果依赖于数据的准确性
- **模型局限**：假设进球事件独立，实际存在相关性
- **特殊因素**：红牌、点球、天气等因素的影响
- **市场因素**：投注市场可能已经反映了模型预测

#### 庄家操作手法

1. **赔率变化信号**
   - **降水**：表示庄家看好某方，可能减少赔付风险
   - **升水**：表示庄家不看好某方，可能引导资金走向
   - **变盘**：重大调整，可能反映伤停或内幕信息
2. **盘口调整心理**
   - **强队让球盘口上升**：可能诱上，强队实际困难
   - **强队让球盘口下降**：真实看好，保护上盘资金
   - **平手盘口向主队倾斜**：主队可能获得优势
3. **资金流向分析**
   - 观察赔率变化与市场资金的配合
   - 识别热门方和冷门方
   - 寻找庄家反向操作的机会

#### 庄家博弈模型

```
庄家目标：最大化利润，最小化风险

1. 平衡两边资金
   - 如果两边资金平衡，庄家无论结果都能稳定盈利
   - 庄家通过调整赔率和盘口引导资金流向

2. 信息优势
   - 庄家拥有更全面的信息（伤病、战意等）
   - 赔率变化反映庄家的信息优势

3. 概率校准
   - 庄家赔率 = 真实概率 + 庄家佣金（ margin）
   - 识别被高估或低估的赔率

4. 心理博弈
   - 利用投注者的心理偏差（热门偏差、情绪投注）
   - 设置陷阱诱导错误判断
```

#### 实战分析步骤

1. **基础概率计算**
   ```
   - 计算各结果隐含概率
   - 对比历史相同赔率的结果概率
   - 识别赔率与基本面的偏离
   ```
2. **盘口合理性验证**
   ```
   - 计算理论盘口（基于球队实力差）
   - 对比实际盘口
   - 分析偏离原因
   ```
3. **资金流向判断**
   ```
   - 观察赔率变化方向
   - 判断热门方和庄家真实看法
   - 寻找反向投注机会
   ```
4. **综合结论**
   ```
   - 赔率支持的方向
   - 盘口支持的方向
   - 庄家心理预期
   - 给出最终判断
   ```

### 7.4 主裁判执法风格分析

#### 裁判类型分类

1. **保守型裁判**
   - 场均黄牌数：< 3.5张
   - 场均红牌数：< 0.1张
   - 判罚尺度宽松，鼓励身体对抗
   - 适合防守强硬或战术犯规较多的球队
2. **标准型裁判**
   - 场均黄牌数：3.5-4.5张
   - 场均红牌数：0.1-0.2张
   - 判罚尺度适中
   - 适合大多数比赛
3. **开放型裁判**
   - 场均黄牌数：> 4.5张
   - 场均红牌数：> 0.2张
   - 判罚尺度严格，鼓励进攻
   - 适合进攻型球队，大球概率高

#### 裁判风格分析方法

1. **历史执法数据**
   - 查询裁判执法记录（whoscored、fbref等）
   - 统计场均黄牌、红牌、点球数
   - 分析执法稳定性
2. **比赛类型适配**
   - 杯赛 vs 联赛：不同比赛类型执法尺度不同
   - 德比战：通常执法更严格
   - 保级/争冠战：关注度高的比赛可能更严格
3. **主客场因素**
   - 裁判对主队的执法记录
   - 主场哨存在吗？数据分析验证
   - 客场球队需要更保守的战术

#### 裁判因素影响评估

```
1. 进攻型球队 + 开放型裁判 = 大球概率↑
2. 防守型球队 + 保守型裁判 = 小球概率↑
3. 粗犷打法 + 标准型裁判 = 红黄牌风险↑
4. 关键球员 vs 严格裁判 = 伤病/停赛风险↑
```

#### 裁判分析步骤

1. **收集裁判数据**
   - 姓名、国籍、执法场次
   - 场均黄牌、红牌、点球
   - 本赛季执法数据
2. **风格判断**
   - 根据数据判断裁判类型
   - 分析执法稳定性
   - 了解特殊偏好（如对某球队特别严格）
3. **影响评估**
   - 对进攻/防守策略的影响
   - 对大小球的影响
   - 对关键球员的风险评估
4. **策略建议**
   - 根据裁判风格调整战术建议
   - 大小球判断参考
   - 红黄牌风险提示

### Python示例：泊松分布预测

```python
import math

def poisson_probability(lambda_rate, k):
    """
    计算泊松分布概率
    :param lambda_rate: 预期进球数（λ）
    :param k: 进球数
    :return: 概率
    """
    return (lambda_rate ** k * math.exp(-lambda_rate)) / math.factorial(k)

def calculate_expected_goals(attack_strength, defense_strength, league_avg, home_advantage=1.1):
    """
    计算预期进球数
    :param attack_strength: 进攻能力
    :param defense_strength: 防守能力
    :param league_avg: 联赛平均进球
    :param home_advantage: 主场优势系数
    :return: 预期进球数
    """
    return attack_strength * defense_strength * league_avg * home_advantage

def predict_over_under(home_lambda, away_lambda, line=2.5):
    """
    预测大小球概率
    :param home_lambda: 主队预期进球
    :param away_lambda: 客队预期进球
    :param line: 大小球盘口
    :return: 大小球概率
    """
    total_lambda = home_lambda + away_lambda
    over_prob = sum(poisson_probability(total_lambda, k) for k in range(int(line) + 1, 10))
    under_prob = sum(poisson_probability(total_lambda, k) for k in range(int(line) + 1))

    return {'over': over_prob, 'under': under_prob, 'line': line}
```

## 爆冷案例分析系统

爆冷（Upset）是足球预测中最具挑战性的部分，因为弱队战胜强队往往有深层原因。本系统通过多维度分析识别爆冷信号，提高预测准确率。

### 爆冷识别核心框架

#### 1. 爆冷定义与分类

**爆冷等级分类**：

| 爆冷等级 | 定义 | 典型赔率差 | 发生概率 |
|---------|------|-----------|---------|
| 一级爆冷（超级冷门） | 降级区球队击败争冠队 | 赔率差 > 3.0 | 约2-3% |
| 二级爆冷（重要冷门） | 保级队击败欧战区队 | 赔率差 2.0-3.0 | 约5-8% |
| 三级爆冷（普通冷门） | 中游队击败欧战区队 | 赔率差 1.5-2.0 | 约10-15% |
| 四级爆冷（轻微冷门） | 弱队击败中游队 | 赔率差 1.2-1.5 | 约15-20% |

**爆冷类型识别**：

```
爆冷类型矩阵：
                    球队状态
                低迷    正常    爆发
赔率差 >2.0   战略放弃  状态爆发  超级爆冷
赔率差 1.5-2.0  状态爆发   战术压制  正常波动
赔率差 <1.5    战术压制   正常波动  强势方过热
```

##### 6.1 爆冷预测算法

```python
def predict_upset_probability(match_data):
    """
    预测爆冷概率
    :param match_data: 比赛数据字典
    :return: 爆冷概率和建议
    """
    base_upset_rate = 0.15  # 基础爆冷率15%

    # 计算各项加成
    odds_adjustment = calculate_odds_adjustment(match_data['odds_change'])
    Kelly_deviation = calculate_Kelly_deviation(match_data['Kelly_index'])
    motivation_factor = calculate_motivation_factor(match_data['teams'])
    fatigue_factor = calculate_fatigue_factor(match_data['schedule'])
    home_advantage = calculate_home_advantage(match_data['home_team'])

    # 综合加成
    total_multiplier = (
        odds_adjustment * 0.20 +
        Kelly_deviation * 0.18 +
        motivation_factor * 0.25 +
        fatigue_factor * 0.22 +
        home_advantage * 0.15
    )

    # 计算最终爆冷概率
    upset_probability = min(base_upset_rate * total_multiplier, 0.60)

    # 生成建议
    if upset_probability > 0.30:
        recommendation = "可关注冷门"
    elif upset_probability > 0.20:
        recommendation = "谨慎观望"
    else:
        recommendation = "正常处理"

    return {
        'upset_probability': upset_probability,
        'recommendation': recommendation,

##### 6.2 爆冷信号量化评分

```python
def calculate_upset_score(match_data):
    """
    计算爆冷综合评分（0-100）
    :param match_data: 比赛数据
    :return: 爆冷评分和等级
    """
    score = 0

    # 赔率异常（最高25分）
    if match_data['odds_change'] > 0.20:
        score += 25
    elif match_data['odds_change'] > 0.10:
        score += 15
    elif match_data['odds_change'] > 0.05:
        score += 8

    # 凯利背离（最高20分）
    Kelly_gap = abs(match_data['Kelly_home'] - match_data['Kelly_away'])
    if Kelly_gap > 0.15:
        score += 20
    elif Kelly_gap > 0.10:
        score += 12
    elif Kelly_gap > 0.05:
        score += 6

    # 战意差异（最高25分）
    if match_data['motivation_gap'] > 2.0:
        score += 25
    elif match_data['motivation_gap'] > 1.5:
        score += 18
    elif match_data['motivation_gap'] > 1.0:
        score += 10

    # 体能差异（最高15分）
    if match_data['fatigue_advantage'] > 0.20:
        score += 15
    elif match_data['fatigue_advantage'] > 0.10:
        score += 8

    # 主场优势（最高15分）
    if match_data['home_advantage'] > 0.15:
        score += 15
    elif match_data['home_advantage'] > 0.10:
        score += 8

    # 等级划分
    if score >= 70:
        level = "极高爆冷风险"
    elif score >= 50:
        level = "高爆冷风险"
    elif score >= 30:
        level = "中等爆冷风险"
    else:
        level = "低爆冷风险"

    return {'upset_score': score, 'upset_level': level}
```

def analyze_kelly的一致性(kelly_values):
    """
    分析凯利指数一致性
    :param kelly_values: 多家公司凯利指数列表
    :return: 分析结果
    """
    import statistics
    mean_kelly = statistics.mean(kelly_values)

    }

### Python示例：泊松分布预测

```python
import math

def poisson_probability(lambda_rate, k):
    """
    计算泊松分布概率
    :param lambda_rate: 预期进球数（λ）
    :param k: 进球数
    :return: 概率
    """
    return (lambda_rate ** k * math.exp(-lambda_rate)) / math.factorial(k)

def calculate_expected_goals(attack_strength, defense_strength, league_avg, home_advantage=1.1):
    """
    计算预期进球数
    :param attack_strength: 进攻能力
    :param defense_strength: 防守能力
    :param league_avg: 联赛平均进球
    :param home_advantage: 主场优势系数
    :return: 预期进球数
    """
    return attack_strength * defense_strength * league_avg * home_advantage

def predict_over_under(home_lambda, away_lambda, line=2.5):
    """
    预测大小球概率
    :param home_lambda: 主队预期进球
    :param away_lambda: 客队预期进球
    :param line: 大小球盘口
    :return: 大小球概率
    """
    total_lambda = home_lambda + away_lambda
    over_prob = sum(poisson_probability(total_lambda, k) for k in range(int(line) + 1, 10))
    under_prob = sum(poisson_probability(total_lambda, k) for k in range(int(line) + 1))

    return {'over': over_prob, 'under': under_prob, 'line': line}
```
