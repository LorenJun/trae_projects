## 🚀 使用方式

### 快速开始（推荐）
```bash
cd europe_leagues
python3 prediction_system.py
```

### 命令行使用
```bash
# 生成预测
python3 prediction_system.py enhanced

# 管理比赛结果
python3 prediction_system.py results

# 查看准确率
python3 prediction_system.py show-accuracy
```

### 独立模块使用
```bash
# 只运行增强预测
python3 enhanced_prediction_workflow.py

# 只管理结果
python3 result_manager.py --interactive
```

## 📈 准确率统计维度

| 维度 | 说明 |
|-----|------|
| 整体准确率 | 所有预测的正确率 |
| 联赛准确率 | 按英超/意甲/德甲/法甲/西甲分类 |
| 模型准确率 | 每个模型单独统计 |
| 比分准确率 | 精确预测比分的正确率 |
| 趋势分析 | 最近7天准确率变化 |

## 🎯 完整工作流程

### 1. 预测阶段
```
输入日期 → 多模型预测 → 生成报告 → 保存到历史数据库
```

### 2. 结果管理
```
比赛结束 → 录入结果 → 系统对比 → 标记正确性
```

### 3. 分析阶段
```
收集数据 → 计算准确率 → 生成报告 → 优化模型权重
```
