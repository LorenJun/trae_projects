```

│       │   └── predictions_template.md  # 预测模板
│       └── results/        # 实际结果
│           └── results_template.md      # 结果模板
├── la_liga/                 # 西甲联赛
```

## 使用方法

### 1. 填写预测数据

在比赛前，使用 `predictions_template.md` 文件填写预测数据：

| 字段 | 说明 | 示例 |
|------|------|------|
| 比赛日期 | 比赛日期 | 2025-08-15 |
| 主队 | 主队名称 | 利物浦 |
| 客队 | 客队名称 | 阿森纳 |
| 预测结果 | 预测的比赛结果 | 主胜/平局/客胜 |
| 预测比分 | 预测的具体比分 | 2-1 |
| 预测概率 | 预测结果的概率 | 0.65 |
| 大小球预测 | 大小球预测 | 大球/小球 |
| 让球盘预测 | 让球盘口预测 | 利物浦-0.5 |
| 实际结果 | 比赛结束后填写 | 主胜/平局/客胜 |
| 实际比分 | 比赛结束后填写 | 2-0 |
| 预测是否正确 | 比赛结束后填写 | 是/否 |

### 2. 填写实际结果

比赛结束后，使用 `results_template.md` 文件填写实际结果：

| 字段 | 说明 | 示例 |
|------|------|------|
| 比赛日期 | 比赛日期 | 2025-08-15 |
| 主队 | 主队名称 | 利物浦 |
| 客队 | 客队名称 | 阿森纳 |
| 实际结果 | 实际比赛结果 | 主胜/平局/客胜 |
| 实际比分 | 实际比赛比分 | 2-0 |
| 主队进球 | 主队进球数 | 2 |
| 客队进球 | 客队进球数 | 0 |
| 总进球 | 总进球数 | 2 |
| 比赛状态 | 比赛状态 | 已结束/未开始 |

### 3. 统计预测准确率

使用 `analyze_prediction_accuracy.py` 脚本统计预测准确率：

```bash
# 运行统计脚本
python analyze_prediction_accuracy.py

# 查看生成的报告
cat prediction_accuracy_report.md
```

## 统计指标

脚本会生成以下统计指标：

### 1. 各联赛准确率
- 比赛总数
- 正确预测数
- 准确率
- 主胜预测准确率
- 平局预测准确率
- 客胜预测准确率
- 大小球预测准确率
- 让球盘预测准确率

### 2. 预测类型分析
- 主胜预测准确率
- 平局预测准确率
- 客胜预测准确率
- 大小球预测准确率

### 3. 改进建议
- 各联赛需要改进的方面
- 总体建议

## 示例使用流程

1. **赛前**：填写预测数据到 `predictions_template.md`
2. **赛后**：填写实际结果到 `results_template.md`
3. **统计**：运行 `analyze_prediction_accuracy.py` 生成准确率报告
4. **分析**：根据报告分析预测表现，调整预测模型

## 注意事项

1. **数据格式**：保持表格格式一致，确保数据能被脚本正确读取
2. **数据完整性**：确保每个比赛都有完整的预测和结果数据
3. **及时更新**：比赛结束后及时更新实际结果，保证统计的准确性
4. **备份数据**：定期备份预测和结果数据，防止数据丢失

## 扩展功能

- **自动数据采集**：可以添加脚本自动从网站获取比赛结果
- **可视化分析**：可以添加数据可视化功能，直观展示预测表现
- **模型优化**：根据统计结果优化预测模型参数
- **多赛季对比**：可以扩展支持多个赛季的对比分析
