# 意甲联赛 2025-26 赛季

## 联赛规则

### 基本规则
- **赛制**：20支球队，双循环赛制，共38轮
- **积分规则**：胜3分，平1分，负0分
- **降级规则**：最后3名降级到意乙
- **欧战资格**：
  - 前4名：欧冠小组赛
  - 第5名：欧联杯小组赛
  - 意大利杯冠军：欧联杯小组赛

### 特殊规则
- **战术风格**：注重防守，战术纪律性强
- **换人规则**：每场比赛最多可换5人，分3次完成
- **VAR使用**：全面使用视频助理裁判
- **球场条件**：部分球场设施较老旧，影响比赛质量

### 对分析的影响
- **防守重要性**：防守质量对比赛结果影响更大
- **战术因素**：战术安排和执行力比球员个人能力更重要
- **主场优势**：主场优势明显，尤其是传统强队

## 球员信息获取方法

### 懂球帝（App / 网页）
**路径**：
- 首页 → 联赛（意甲） → 点击任意球队 → 球队阵容

**直接链接**：
- 国际米兰：`https://m.dongqiudi.com/team/1.html`
- 其他球队：将末尾数字改对应球队 ID 即可查所有队

### 使用说明
1. 访问懂球帝网站或App
2. 选择意甲联赛
3. 点击对应球队进入球队页面
4. 查看球队阵容获取最新球员信息
5. 根据获取的信息更新项目中的球员JSON文件
